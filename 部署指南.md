## 部署步骤

### 第一步：创建 Supabase 项目

1. 访问 https://supabase.com 注册/登录
2. 点击 "New Project" 创建新项目
3. 设置项目名称和密码
4. 等待项目创建完成

### 第二步：初始化数据库

1. 进入 Supabase 控制台 → SQL Editor
2. 新建查询，复制 `supabase/schema.sql` 全部内容
3. 点击 "Run" 执行

### 第三步：获取 Supabase 配置

1. 进入 Project Settings → API
2. 复制:
   - **Project URL**: `https://xxxxx.supabase.co`
   - **anon public**: `eyJhbG...`

### 第四步：创建管理员账号

1. 进入 Authentication → Users
2. 点击 "Add user"
3. 输入管理员邮箱和密码

### 第五步：部署到 Vercel

#### 方法1: GitHub + Vercel (推荐)

1. 在 GitHub 创建新仓库 `stockai-website`
2. 推送代码到仓库:
```bash
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/你的用户名/stockai-website.git
git push -u origin main
```

3. 访问 https://vercel.com 登录
4. 点击 "Add New Project" → "Import Git Repository"
5. 选择 `stockai-website` 仓库
6. 配置环境变量:
   - `VITE_SUPABASE_URL`: 你的 Supabase URL
   - `VITE_SUPABASE_ANON_KEY`: 你的 Supabase Anon Key
7. 点击 "Deploy"

#### 方法2: Vercel CLI

```bash
# 安装 Vercel CLI
npm i -g vercel

# 登录
vercel login

# 部署
vercel

# 设置环境变量
vercel env add VITE_SUPABASE_URL
vercel env add VITE_SUPABASE_ANON_KEY

# 重新部署
vercel --prod
```

## 环境变量配置

创建 `.env` 文件:

```env
VITE_SUPABASE_URL=https://your-project.supabase.co
VITE_SUPABASE_ANON_KEY=your-anon-key
```

在 Vercel 部署时，需要在项目设置中添加这些环境变量。

## 常见问题

### Q: 部署后页面空白?
A: 检查环境变量是否正确设置，Supabase 项目是否正常运行

### Q: 无法登录后台?
A: 确认已在 Supabase Auth 中创建用户，邮箱密码正确

### Q: 如何绑定自定义域名?
A: 在 Vercel 项目设置 → Domains 中添加域名并配置 DNS
